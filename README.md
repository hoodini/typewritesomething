Original site: [typewritesomething.com](https://typewritesomething.com)

## Fork Enhancements

This fork adds the following features:

- **Hebrew RTL Support** - Automatically switches to right-to-left mode when typing Hebrew
  - Cursor auto-jumps to right margin when starting Hebrew text
  - Enter key respects RTL mode (new line starts from right)
- **Export as Image** - Export your typewriter page as a clean white PNG image
